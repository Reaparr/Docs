# Support Reaparr

Reaparr is a free and open-source project that takes considerable time and effort to develop and maintain. If you find Reaparr useful and would like to support its continued development, please consider becoming a sponsor!

## Why Donate?

Your support helps with:

- **Ongoing Development** - Adding new features and improvements
- **Bug Fixes** - Maintaining stability and fixing issues
- **Infrastructure Costs** - Hosting, testing environments, and tools
- **Documentation** - Keeping guides and docs up-to-date
- **Community Support** - Responding to issues and helping users

## GitHub Sponsors

You can support Reaparr through GitHub Sponsors, which offers flexible monthly sponsorship options.

[Become a Sponsor on GitHub](https://github.com/sponsors/Reaparr){target="_blank" class="button"}

## Other Ways to Help

If you're unable to donate financially, there are other valuable ways to contribute:

- **Star the project** on GitHub to help increase visibility
- **Share Reaparr** with others who might find it useful
- **Report bugs** and suggest features through GitHub issues
- **Contribute code** by submitting pull requests
- **Improve documentation** to help other users
- **Help others** in discussions and support channels

Every contribution, whether financial or otherwise, is greatly appreciated and helps make Reaparr better for everyone!

## Thank You!

A huge thank you to all our supporters and contributors who make this project possible. Your generosity and support mean the world to us!
