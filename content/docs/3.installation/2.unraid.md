# Installation on Unraid

:::tip{to="https://forums.unraid.net/topic/114103-support-reaparr-the-best-cross-platform-plex-media-downloader-there-is/"}
Unraid support thread
:::

### 1. Ensure you have [Community Applications](https://unraid.net/community/apps?q=reaparr#r) installed.
### 2. Search for Reaparr and click install.

![Alt text](/img/screenshots/demo-10-unraid-ca.avif "Reaparr search")

### 3. Now you can select :latest or :dev as the version you want to install, and click `Install` again.

![Alt text](/img/screenshots/demo-10-unraid-ca-install.avif "Reaparr Unraid Install")

### 4. A Docker Template will be shown, fill this in with the correct values

### 5. Find your user’s PUID and PGID

Run this command in the Unraid web terminal to find your user’s UID and GID:
```bash
id
```

Example output:
```
uid=1000(user) gid=1000(user) ...
```
- `uid=1000` → `PUID=1000`
- `gid=1000` → `PGID=1000`

More info: https://docs.linuxserver.io/general/understanding-puid-and-pgid/

Fill in these values in the template (Click on "Show more settings" to expand the options if you don't see the PUID and PGID fields"):

![Alt text](/img/docs/unraid-install/unraid-more-settings.avif "Reaparr Unraid More Settings")

### 6. Once you have filled in the template, click `Apply` to start the container.

### 7. Done!

:::tip{to="/docs/configuration/set-up"}
Continue with the Reaparr setup guide to connect your Plex account and configure your settings."
:::
